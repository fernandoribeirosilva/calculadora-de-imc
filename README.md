# Calculadora de IMC

Projeto em React + Typescript fazendo uma calculador de IMC.

Este projeto foi feito no curso da [B7Web](https://b7web.com.br).

## Instalação
 ```npm
 npm install
 ```

 ## Para rodar 
 ```npm
npm run start
 ```
